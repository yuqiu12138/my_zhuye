自学分享

——下面分享一些有关我学习前端的心得，以及给一些想去学习前端的童鞋的建议。


一、要明确自己的目标
首先，要确定自己的目标，确定自己感兴趣的方向，就以我来说，在学校期间是以JAVA语言开发为主，因为自己对APP软件开发有着很浓厚的兴趣，一度选择android软件开发。但是如今H5的快速发展，前端的各种框架也能实现类似原生android软件的效果，从而使我确定自己想从事web前端开发。

二、快速测试自己是否适合
明确自己学习目标后，建议寻找一些快速入门的学习资料（最好是在学习过程中可以进行对应小项目的编写）。看完这些资料并尝试过完成小项目后，如果还对所学的感兴趣，那么就可以开始正式学习了。（这里主要是想测试一下自己是否适合所学的东西，减少学习成本）

三、学习基础内容
这里基本上就是理解原理，记一些基础知识点。（学习基础总是枯燥的，但是只要我们充分掌握基础知识，那么在之后的学习才会更顺利），我自己是通过在图书馆借书，已经在各种视频网站上进行看视频进行学习。

四、学习进阶内容并尝试完成小项目
充分了解和学习完基础知识之后，我们可以看一些关于小项目的资料。尝试运用我们所学的知识,完成一些小项目。（通过完成项目来测试自己所学的内容是不错的方法）

五、整合所学的不同知识点尝试完成一个完成的项目
经过一段时间的学习，我们所接触的知识点会越来越多。这时我们可能开始会对一些知识点产生混乱，所以我们要善于整理所学过的内容，找共同点和差异点加以分析。并尝试运用所学的知识点。独立完成项目（这里建议各位在做项目的时候尽量能后少看资料，先自己尝试解决。如果解决不了看资料）。

六、最后建议
IT行业是一个不断发展的行业，首先要确定自己的心态，做好不断去学习的准备，我们有时候去学的不仅仅是技术，而是别人一种开发的思维，这样才能使自己更进一步发展。

Html & Css & Bootstrap


—— 简介
Html:超文本标记语言（英语：HyperText Markup Language，简称：HTML）是一种用于创建网页的标准标记语言。
Css:指层叠样式表 (Cascading Style Sheets)。
Bootstrap:是一个用于快速开发 Web 应用程序和网站的前端框架。
—— 个人理解
Html:用于创建网页框架的编程语言（比如：建筑地基框架）。
Css:用于操控Html框架的布局以及如何显示的编程语言（比如：建筑装修后的样子）。
Bootstrap:网站框架搭建的工具库（比如：建筑所使用的材料）
Html5/Css3:版本不同，增加、减少或者优化功能。（比如：不同的施工队）。
